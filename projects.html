Mingle Beyond - Social Platform for Recent Graduates

Mingle Beyond is a comprehensive web platform designed to help recent college graduates stay connected and supported during the major transition from college to professional life. The platform addresses feelings of isolation and uncertainty by providing a space where users can meet new people, explore shared interests, and build genuine connections within a supportive community.

Duration: Fall 2024 - Spring 2025
Team: 3 Members (Team 27)
Course: INFO Capstone 2025

Key Features:

  • Personalized user profiles with photos and personal information
  • Event RSVP system with attendee visibility
  • Friend request system and private messaging functionality
  • Hobby-based resource discovery and activity recommendations
  • Featured events and curated content on homepage
  • Google Maps integration for location-based features
  • Google Login authentication for seamless user experience

Design Philosophy:

The platform emphasizes user-friendly design, accessibility, and personal expression. Built with a focus on simplicity and inclusivity, Mingle Beyond enables users to navigate with ease and truly make the platform their own. The goal is to make social support easier to find and more authentic for young adults navigating life after college.

Technologies Used:

PHP MySQL Google Maps API Google Login HTML5 CSS3 JavaScript Responsive Design
